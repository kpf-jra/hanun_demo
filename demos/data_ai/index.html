예시 질문

이 데모는 어떻게 동작하나요?

엑셀 표를 그대로 AI에 넘기지 않고, 항목·연도·지표·수치가 분리된 JSON 레코드로 바꿔 전달합니다. 질문마다 관련 행만 골라 Gemini에 넘기고, 표 밖 질문은 거절합니다.

현재

  • 표 렌더링 + 기본 요약(고정) · 「다시 분석」으로 AI 요약 재생성
  • 질문 유형별 필터(연도 범위, 비교, 소계 등)와 프롬프트 분기
  • 표 관련 여부 규칙 판별 + Gemini 모델 선택

앞으로 개선할 수 있는 방향

  • 단순 수치 조회는 코드가 직접 답하고, 추이·비교만 AI가 설명 (환각 감소)
  • 증감·변화폭 등 파생 지표를 미리 계산해 함께 제공
  • AI 답변의 숫자가 표에 실제 있는지 검증하는 단계 추가
