KOSIS 통계 API 데모

← Lab 홈

KOSIS OpenAPI로 한국언론진흥재단 통계 중 API 수치 조회가 가능한 신문산업 표 16개를 바로 보여 줍니다. KOSIS 목록에는 370여 개가 있으나, OpenAPI로 수치가 열린 표는 이 16개뿐입니다.

설정 필요: KOSIS 회원가입 후 활용신청으로 인증키를 발급받고, 프로젝트 루트 .env에 KOSIS_API_KEY=발급키를 넣은 뒤 node server.mjs를 재시작하세요. 회사망 SSL 오류 시 GEMINI_INSECURE_SSL=1도 함께 설정할 수 있습니다.
