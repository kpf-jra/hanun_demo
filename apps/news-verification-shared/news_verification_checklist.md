# 부록 2. 뉴스 생산 과정의 사실 확인 및 검증 체크리스트

이 체크리스트는 뉴스 생산의 각 단계(취재, 보도, 게이트키핑)와 특수 보도(탐사보도, 사진·영상, 온라인 정보) 상황에서 사실 관계를 확인하고 검증하기 위한 항목들로 구성되어 있습니다.

---

## 1. 취재 단계

| 항목 | 확인할 내용 | 체크 |
| :--- | :--- | :---: |
| **직접 취재** | 직접 사실을 수집하고 정보를 검증했는가? | |
| | 보도하려는 사실과 관련된 모든 취재 대상자에게 대면 또는 전화 통화 등을 통해 사실관계를 직접 확인했는가? | |
| | 보도하려는 사실과 관련된 모든 취재 대상자에 대한 모든 연락 시도와 응답 여부를 기록해 보관했는가? | |
| | 기사에 중요하게 활용되는 자료(통계, 학술논문 등)의 원천 자료를 직접 읽어보고 확인했는가? | |
| **현장 취재** | 현장이 존재하는 사안의 경우 현장을 방문하여 확인했는가? | |
| | 가능한 한 현장의 많은 관계자와 직접 대화했는가? | |
| | 진술이나 문건을 현장에서 교차 확인했는가? | |
| | 직접 현장 확인이 불가능한 경우, 현장 목격자나 사안을 직접 경험한 1차 당사자와 인터뷰했는가? | |
| **밀착 취재** | 공적 질문에 답할 책임이 있는 핵심 대상자에게 적극적으로 접근해 답변을 구했는가? | |
| | 사실 관계 확인 방법이 부족한 경우, 관련 인물의 폭을 최대한 넓혀 접촉했는가? | |
| | 보도하려는 사실과 관련된 자료의 폭을 최대한 넓혀 수집했는가? | |
| **반론** | 예상되는 반론과 반대 사실에 대해서도 가능성을 열어두고 취재했는가? | |
| | 취재원에게 불리하거나 비판적인 내용을 다룰 경우, 반론 대상자에게 입증·해명에 필요한 정보를 구체적으로 전달했는가? | |
| | 취재원에게 불리하거나 비판적인 내용을 다룰 경우, 합리적으로 응답할 수 있는 충분한 시간을 주었는가? | |
| | 동의를 얻지 않은 방문취재를 하는 경우 중대한 의혹 대상자가 반론 기회를 반복적으로 거부하는 등의 사유가 존재하는가? | |
| | 취재 대상의 주장뿐 아니라, 그에 반대되는 입장에서도 의심의 시선으로 사실을 검증했는가? | |
| | 기자가 전제한 기본 관점이나 취재 방향에 오류 가능성이 있음을 인정하고, 스스로 점검했는가? | |
| **사실 검증** | 기사로서 형식적 요건과 내용적 완전성을 갖추기에 충분할 정도로 사실 확인을 했는가? | |
| (교차 검증, 복수 취재) | 특정 개인, 단체, 기관의 관점을 비판적으로 검토했는가? | |
| | 취재 과정에서 알게 된 내용을 복수의 독립된 취재원을 통해 교차 확인했는가? | |
| | 단일 취재원에만 의존하고 있지는 않은가? | |
| | 단일 취재원의 정보를 뒷받침할 다른 취재원나 근거 자료를 확보했는가? | |
| | 단일 취재원 사용이 불가피한 경우 데스크에게 보고하고 승인을 받았는가? | |
| | 익명 단일 취재원을 사용하고 있지는 않은가? | |
| | 익명 단일 취재원 사용이 불가피한 경우, 이를 정당화할 근거를 데스크에게 제시하고 승인을 받았는가? | |
| | 취재원의 진술을 뒷받침할 신빙성 있는 근거 자료(문서·기록·데이터)를 확보했는가? | |
| | 문서·기록을 현장 취재로 입증할 수 있는지 확인했는가? | |
| | 현장 관찰·목격, 문서·데이터 분석, 인물 인터뷰 중 두 가지 이상의 방법을 적용했는가? | |
| | 취재원의 진술에 반대되는 견해나 상황이 있는지 파악했는가? | |
| | 특정 '단골 전문가'에게 반복적으로 의존하고 있지는 않은가? 새로운 전문가 집단을 발굴하려고 노력했는가? | |
| | 사안에 직접적 영향을 받는 이해당사자들의 목소리를 모두 반영했는가? | |
| | 사안에 간접적 영향을 받는 이해당사자들의 목소리를 최대한 반영했는가? | |
| **발언·발표·보도자료** | 취재원의 발언과 보도자료에 나오는 내용을 별도의 출처를 통해 확인했는가? | |
| | 중요하게 다뤄져야 할 내용이 보도자료에서 설명되지 않거나 누락되지 않았는지 확인했는가? | |
| | 보도자료에서 지표를 왜곡하거나 중요한 정보를 의도적으로 누락하지 않았는가? | |
| | 발표 내용이 실제로 작동하는지 사안에 직접 관련된 당사자들에게 확인했는가? | |
| | 보도자료에서 실효성에 의문이 가는 부분을 집중적으로 검증했는가? | |
| | 필요시 해당 사안에 정통한 전문가나 이해당사자들의 의견을 반영했는가? | |
| | 수사 중인 사건을 무죄 추정의 원칙에 따라 교차 검증을 통해 확인했는가? | |
| | 설명 근거가 빈약하거나 모순이 있다고 판단되면 발언 당사자에게 근거를 요구하고 집요하게 물었는가? | |
| **제보·의혹 확인** | 제보·의혹을 제기한 동기와 맥락을 파악했는가? | |
| | 직접 경험자·관계자 등 1차 취재원을 통해 기초 사실을 교차 확인했는가? | |
| | 제보 내용을 복수의 다른 취재원을 통해 교차 검증했는가? | |
| | 개인의 비위·일탈이나 단일 사건을 넘어 제도와 체계 등 구조적 문제로 파악했는가? | |
| **타 매체 보도 확인** | 타 매체 보도를 그대로 받아쓰지 않고 독자적으로 확인했는가? | |
| | 타사 보도를 검증할 때 원 출처(original sources)를 상대로 사실 부합 여부를 확인했는가? | |
| | 독자적으로 확인하지 못한 경우 타 매체 출처를 밝히고 활용했는가? | |
| | 타사·외신 보도에서 활용한 취재원에 접근할 수 있는가? | |
| | 타사·외신 보도에서 활용한 문서, 영상 등 자료를 확보할 수 있는가? | |
| **AI 확인** | 생성형 AI를 활용해 파악한 정보를 교차 확인을 비롯한 인간의 검증을 거쳤는가? | |
| **취재 윤리** | 문서를 훔치거나 무단으로 가져가는 방식으로 자료를 취득하지 않았는가? | |
| | 동의 없이 타인의 컴퓨터나 디지털 정보에 접근하지 않았는가? | |
| **취재원과의 합의** | 취재원과 신원 표현 방식(실명/익명, 직책 언급 여부 등)을 명확하게 합의했는가? | |
| | 익명을 허용한 경우, 익명이 어디까지 적용되는지 취재원과 상호 명확히 이해했는가? | |
| | 방송의 경우, 얼굴과 목소리 처리 방법을 취재원에게 명확하게 알리고 합의했는가? | |
| | 인용 방식(온더레코드/백그라운드/딥백/오프더레코드)을 명확하게 합의했는가? | |
| | 취재원이 기자라는 사실과 보도 목적임을 명확히 인지하고 있는가? | |
| | 취재원과의 합의 과정을 투명하게 기록으로 남겼는가? | |
| | 익명 협의 과정을 정확하고 시의적절하게 기록(가능하면 녹음)으로 남겼는가? | |
| **기록·보관** | 취재 내용을 시차를 두지 않고 즉시 기록했는가? | |
| | 취재 기록을 체계적으로 정리해 보관했는가? 디지털 형태(파일, 클라우드 저장소 등)로도 보관했는가? | |
| | 확보한 문서, 배경 메모, 이메일, 서면 자료를 모두 보관했는가? | |
| | 취재원과 주고받은 대화를 어떤 형태든 전부 기록해 보관했는가? | |
| | 취재 기록을 필요할 경우 다른 팀원이 확인할 수 있도록 체계적으로 보관했는가? | |
| **비밀 녹화·녹음** | 비밀 녹화·녹음을 방송에 사용해야 한다면 데스크나 법무팀과 사용 목적과 정당성을 논의했는가? | |
| | 비밀 녹화·녹음을 방송에 사용할 경우 범죄나 비리 의혹 규명 등 반드시 정당화할 수 있는 근거가 있는가? | |

## 2. 보도 단계

| 항목 | 확인할 내용 | 체크 |
| :--- | :--- | :---: |
| **취재 동기의 공개** | 왜 이 내용이 기사로 다뤄져야 하는지 취재와 보도의 동기를 가능한 범위에서 설명했는가? | |
| **취재 과정의 공개** | 취재 과정에서 검증을 위해 시도한 노력과 방법을 소개했는가? | |
| | 취재의 핵심 근거가 된 자료를 확보한 경로, 입수 경위를 가능한 범위에서 설명했는가? | |
| | 취재원의 발언이 나오게 된 맥락(기자회견, 전화 인터뷰, 대면 대화 등)을 설명했는가? | |
| | 사실 확인을 위해 접촉한 취재원의 수, 취재 기간, 취재에 활용한 방법 등을 밝혔는가? | |
| | 적절하다고 판단되면, 취재 과정에서 발생한 어려움이나 딜레마 상황도 드러냈는가? | |
| | 방송뉴스의 경우 기자 출연 코너에서 취재 과정을 설명했는가? | |
| | 대상자가 답변을 거부하거나 무응답인 경우, 대상자의 기존 입장을 반영했는가? | |
| | 반론권을 부여한 취재원이 응답을 거부하면 거부했다는 사실을 명시했는가? | |
| | 진위가 아직 확인되지 않은 내용에 대해서는 아직 '검증되지 않았다'는 표시를 했는가? | |
| | 일방적 주장이나 폭로를 아직 확인하지 못한 경우 ‘근거가 확인되지 않은’이라고 표시했는가? | |
| | 타사 보도를 확인하지 못한 상태로 인용 보도하는 경우, '우리는 독자적으로 확인하지 못했다'는 한계를 분명히 밝혔는가? | |
| | 취재 과정에서 생성형 AI 도구를 상당한 정도로 활용했다면 사용 이유와 방법을 투명하게 공개했는가? | |
| **취재원의 실명 제시** | 취재원의 실명과 직함을 가능한 한 공개했는가? | |
| | 1명의 취재원에게서 들은 내용을 복수의 취재원에게서 확보한 것처럼 표현하지 않았는가? | |
| | 집단 취재원의 형태(예: '전문가들은', '법조계 안팎에선')를 사용하는 경우 반드시 복수의 취재원에게 확인했는가? | |
| | 독자적 확인 여부와 관계없이, 타 매체가 먼저 보도한 사실을 인용할 때 출처를 명확히 밝혔는가? | |
| | 타 매체가 출처일 경우, '한 신문에 따르면', '한 방송에 따르면' 같은 식으로 불투명하게 출처를 밝히지 않고 이름을 명확히 밝혔는가? | |
| | TV·라디오 방송 프로그램에 출연한 인물의 인터뷰를 인용하는 경우 해당 프로그램명을 명시했는가? | |
| | 가능하다면 타사 기사의 링크도 삽입했는가? | |
| | 필요하고 적절한 경우, 정보의 출처가 되는 원자료(예: PDF 파일, 링크 등)를 기사에 함께 제공했는가? | |
| **익명 취재원의 사용** | 취재원을 익명으로 표기하기 전에 사용 조건에 맞는지 데스크와 상의했는가? | |
| | 익명을 요구한 출처를 제외하고는 현실적으로 해당 정보를 입수할 방법이 없는가? | |
| | 익명 취재원이 신뢰할 만하며 해당 정보를 직접적으로 알고 있는 위치에 있는가? | |
| | 익명 취재원의 발언이 의견이 아닌 사실 정보인가? | |
| | 익명으로 타인을 비방하는 인용문을 사용했는가? | |
| | 익명 취재원으로 표기할 경우 그럴 만한 이유를 기사에 명시했는가? | |
| | 타사 보도의 익명 취재원을 직접 접촉하지 못한 상태로 보도하는 경우, 타사 보도가 출처임을 밝히고 인용했는가? | |
| **사실과 의견의 구분** | 형용사, 부사, 보조사 등 독자의 해석에 영향을 줄 수 있는 표현은 사용을 자제했는가? 사용한 경우, 객관적 근거를 함께 제시했는가? | |
| | 형용사, 부사, 보조사 등 의견으로 의심되는 표현을 사용한 경우, 객관적 근거를 기사 내에 제시했는가? | |
| | ‘말했다’, ‘밝혔다’, ‘전했다’ 등 중립적이고 객관적인 술어를 사용했는가? | |
| | ‘지적된다’, ‘알려졌다’, ‘추정된다’ 등 무주체 피동형 표현을 사용하지 않았는가? | |
| | 기자나 매체의 해석, 평가, 판단이 개입된 문장은 명확히 구분되거나 별도로 표시되었는가? | |

## 3. 게이트키핑 단계

| 항목 | 확인할 내용 | 체크 |
| :--- | :--- | :---: |
